# Profiles

This directory contains built-in scan and film profiles for Positize.

## Profile Types

### Scan Profiles
Define capture source characteristics (DSLR/mirrorless, flatbed scanner) and typical processing parameters.

### Film Presets
Film-specific settings including base offsets, color matrices, and tone curves for accurate negative conversion.

## Examples

- `scan/dslr-copy.yml`: DSLR camera scan profile
- `scan/flatbed.yml`: Flatbed scanner profile
- `film/generic-color-negative.yml`: Generic color negative preset
- `film/generic-bw.yml`: Generic black and white negative preset
